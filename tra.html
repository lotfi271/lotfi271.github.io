/ Comment ça marche

Trois étapes.
Un pont entre deux mondes.

01
Capture & Détection
MediaPipe extrait 21 points de repère de la main (landmarks 3D) en temps réel à 30fps depuis votre webcam.
02
Reconnaissance des signes
Un réseau de neurones LSTM analysé sur 50 000 signes LSF identifie le signe et son contexte temporel.
03
Traduction & Synthèse
Le texte est généré en français et peut être synthétisé en voix. Le flux inverse génère un avatar signeur 3D.
/ Fonctionnalités

Tout ce dont la communauté sourde a besoin

Conçu avec des associations sourdes, testé sur des vrais locuteurs LSF. Pas une traduction de l'ASL. Un système natif.

🤚
Signes complets LSF
Pas seulement l'alphabet dactyologique — reconnaissance des signes lexicaux complets, expressions et paramètres (chérème, emplacement, mouvement).
⚡
Temps réel <80ms
Traitement sur GPU local ou CPU moderne. Aucun cloud requis. Vos données restent sur votre appareil — conformité RGPD native.
🗣️
Synthèse vocale
Voix naturelle en français avec Kokoro-TTS, open source. Adapté aux contextes professionnels : médecin, administration, école.
🧑‍💻
Avatar signeur 3D
Texte ou voix en entrée → avatar LSF animé en sortie. Idéal pour les entendants qui veulent communiquer avec la communauté sourde.
🐧
Compatible Linux
Tournez sur Ubuntu, Debian, Fedora. Prêt pour les futurs postes Linux de l'État français. Python 3.10+, pas de dépendances propriétaires.
🔓
100% Open Source
Licence MIT. Modèles sur HuggingFace. Données d'entraînement issues de l'Université Grenoble-Alpes et associations sourdes partenaires.

RECHERCHE
PYTHON · traducteur_sourd.py 
# Installation rapide
pip install traducteur-sourd

# Usage basique
from traducteur_sourd import LSFTranslator

translator = LSFTranslator(
  model="lsf-v2.1",
  output="text+voice",
  lang="fr"
)

# Démarrer la traduction temps réel
with translator.webcam() as cam:
  for sign in cam.stream():
    print(sign.text, sign.confidence)
    # → "Bonjour" 0.94
